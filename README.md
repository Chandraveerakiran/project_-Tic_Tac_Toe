# Tic Tac Toe

A two-player Tic Tac Toe game with a desktop GUI, built entirely in Python.

---

## Built With

- **Python 3**
- **Tkinter** — standard GUI library, no external dependencies

---

## Getting Started

```bash
git clone https://github.com/your-username/tic-tac-toe.git
cd tic-tac-toe
python tic_tac_toe.py
```

That's it. No pip installs, no virtual environments.

---

## How It Works

- 3×3 clickable grid, two players alternate turns
- X renders in **blue**, O renders in **red**
- Detects all 8 win conditions automatically
- Board locks on win — hit Reset to replay
